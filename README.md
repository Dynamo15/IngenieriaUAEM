Execution Disclaimer

This project cannot be executed in isolation.

For the full demonstration to work, it would require:
- A second external system
- A remote controlled environment (e.g., Kali Linux)
- Additional backend components not included in this repository

All such components were intentionally excluded.
This repository contains only a conceptual and educational version.
# Web Security Awareness – Camera Exploit Demo

 Descripción
Proyecto académico desarrollado en entorno controlado con fines educativos,
orientado a demostrar vulnerabilidades de seguridad presentes en páginas web falsas.

Objetivo
Concientizar sobre:
- Riesgos de ingeniería social
- Uso indebido de permisos del navegador
- Importancia de verificar sitios web

Tecnologías usadas
- HTML
- CSS
- JavaScript
- Kali Linux (entorno de recepción de datos)

Consideraciones éticas
Por razones éticas y legales:
- El código ofensivo no se publica
- El proyecto no está desplegado
- No existe demo funcional

Contexto académico
Proyecto realizado como parte de una práctica universitaria
para explicar vulnerabilidades web y buenas prácticas de seguridad.
